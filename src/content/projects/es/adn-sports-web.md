## Description / Descripción

Este proyecto es un sitio web moderno y optimizado para un centro deportivo, desarrollado utilizando el framework **Astro** para garantizar el máximo rendimiento y una excelente optimización para motores de búsqueda (SEO). El objetivo principal era crear una presencia digital rápida y eficaz que cubriera las necesidades del cliente.

El sitio cuenta con múltiples páginas que detallan exhaustivamente las instalaciones del centro y la variedad de servicios ofrecidos, proporcionando a los usuarios toda la información necesaria de manera clara y accesible. Al estar construido con una arquitectura de islas y generación estática (SSG), ofrece tiempos de carga mínimos y una experiencia de usuario fluida, sin depender de un backend complejo.

## Key Features / Características Principales

- **Optimización SEO Avanzada**: Estructura y metaetiquetas optimizadas para mejorar la visibilidad en buscadores.
- **Máximo Rendimiento**: Cargas ultra-rápidas gracias a la arquitectura de Astro y la ausencia de JavaScript innecesario.
- **Diseño Multi-página**: Navegación clara a través de distintas secciones dedicadas a servicios e instalaciones.
- **Showcase de Servicios**: Presentación detallada de la oferta deportiva y actividades del centro.
- **Sin Backend**: Arquitectura simplificada y segura, reduciendo costes de mantenimiento y puntos de fallo.
