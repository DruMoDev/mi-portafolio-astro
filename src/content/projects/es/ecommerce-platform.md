## Descripción del Proyecto

Una plataforma de comercio electrónico completa desarrollada desde cero. Incluye autenticación de usuarios, catálogo de productos con filtros avanzados, carrito de compras persistente, integración con Stripe para pagos seguros, y un panel de administración completo.

## Características principales

- Autenticación JWT con refresh tokens
- Búsqueda y filtrado de productos en tiempo real
- Carrito de compras con persistencia en base de datos
- Pasarela de pagos Stripe
- Panel de administración con estadísticas
- Sistema de notificaciones por email
- API RESTful documentada con Swagger
