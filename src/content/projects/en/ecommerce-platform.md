## Project Description

A complete e-commerce platform built from scratch. Includes user authentication, product catalog with advanced filters, persistent shopping cart, Stripe integration for secure payments, and a comprehensive admin dashboard.

## Key Features

- JWT Authentication with refresh tokens
- Real-time product search and filtering
- Persistent shopping cart
- Stripe payment gateway integration
- Admin dashboard with analytics
- Email notification system
- RESTful API documented with Swagger
